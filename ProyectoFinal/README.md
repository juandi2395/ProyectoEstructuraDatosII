# ProyectoEstructuraDatosII
Sistema de gestión de biblioteca desarrollado en C++ con estructuras de datos avanzadas y persistencia en SQL.

Sistema de Gestión de Biblioteca es un proyecto académico desarrollado en C++ cuyo objetivo principal es implementar y aplicar estructuras de datos avanzadas dentro de un escenario de administración de información. El sistema permite gestionar libros, usuarios, préstamos y recomendaciones mediante estructuras especializadas, complementadas con una base de datos SQL para la persistencia de la información.

El proyecto está diseñado siguiendo una arquitectura modular basada en programación orientada a objetos, separando las responsabilidades del sistema en capas independientes para facilitar el mantenimiento, la escalabilidad y la reutilización del código.
